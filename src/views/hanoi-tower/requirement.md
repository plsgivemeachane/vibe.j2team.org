**Tower of Hanoi – Multi-Level Puzzle Game**

The **Tower of Hanoi** game challenges players to solve a classic logical puzzle by moving disks between three towers. The goal is to transfer the entire stack from the starting tower to the target tower while following strict rules: only one disk can be moved at a time, and a larger disk can never be placed on top of a smaller one.

This version introduces a **multi-level progression system** that gradually increases difficulty. Each level starts with a different number of disks, requiring more strategic planning and a greater number of moves to complete.

**Game Features**

* **Multiple Levels:** Start with simple puzzles (3 disks) and progress to more complex challenges with additional disks.
* **Increasing Difficulty:** Each level requires more moves and deeper planning.
* **Move Counter:** Track the number of moves used and compare with the optimal solution.
* **Hints System (optional):** Players can receive guidance when stuck.
* **Time Tracking:** Measure how quickly each level is solved.
* **Progression System:** Unlock higher levels after completing previous ones.

**Objective**
Solve each level using the minimum number of moves possible. As the number of disks increases, the puzzle becomes exponentially more challenging, encouraging players to improve their logical thinking and problem-solving skills.

**Target Experience**
The game is designed to help players develop  **algorithmic thinking, patience, and strategic planning** , making it both an educational and entertaining puzzle experience.
